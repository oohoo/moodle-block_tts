--------------------------------------------------------------------------------
-------------------------- OOHOO - TTS - Text To Speech ------------------------
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
Description
--------------------------------------------------------------------------------

OOHOO Text to Speech is a Moodle block that gives the possibility to read out loud the content of a resource.


--------------------------------------------------------------------------------
Prerequisites
--------------------------------------------------------------------------------

The Apache server must have an Internet connectivity with a well configurated
gateway. Because it is the server that grab sound from Google or Microsoft (To
keep them in cache).

--------------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------------

 1. Rename the folder to 'tts'
 2. Copy the folder tts to moodle/blocks
 3. Install the plugin
 4. Configure the TTS options in the settings

Note: You can test you server connectivity by accessing to the page: 
http://yourmoodleurl/blocks/tts/tts_test.php
